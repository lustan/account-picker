
# 项目概览
Account Picker 可帮助开发者和 QA 在多个环境（dev/uat/prod）中快速登录。你可以为每个环境保存多个账号，然后一键选择并自动填充用户名和密码。

# 🎯 功能特性
1. 一键自动填充，快速登录
2. 多环境管理（URL + CSS 选择器）
3. 使用标签按环境筛选账号
4. 支持行内编辑、分页和删除
5. 支持 JSON 导入/导出，便于备份与共享
6. 支持密码可见性切换
7. 可选“禁用自动填充”，避免浏览器建议干扰

# ✨ 使用方式
1. 添加环境（包含 URL，以及可选的用户名/密码输入框选择器）。
2. 添加账号并分配到对应环境。
3. 在登录页打开插件，选择一个账号即可自动填充。

# 🛶 隐私说明
所有数据都仅存储在你的浏览器本地（Chrome storage）。
不会将任何账号数据发送到服务器。
相关权限仅用于保存设置以及在你配置的网站上填充表单。
